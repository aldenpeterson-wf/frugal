### Frugal Python Example
=====

1. Highly recommend making a virtualenv with virtualenvwrapper:
    ```
    mkvirtualenv frugal-example -a $PWD
    ```

1. Install deps:
    ```
    pip install -Ur requirements.txt
    ```

1. Start up server:
    ```
    python main.py -server
    ```

1. Run client:
    ```
    python main.py
    ```
